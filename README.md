## Info
A full-stack app to keep the name of invitees and their RSVP status.


## CRUD API

Add new invitee:
```
POST   /api/invitees
```

Get the list of invitees:
```
GET   /api/invitees
```

## Installing Dependencies

From within the root directory:
```
npm install
```

Run the setup to get everything up and running
```
npm run setup
```

To run server
```
npm start
```

To run tests
```
npm test
```
